Novas USs:  

Ao se finalizar uma venda, ela venda deve ser salva em um arquivo. 
Catálogo também deve ser lido/salvo de um arquivo. 

Usuário cadastra venda com itens e, depois de finalizá-la, emite nota fiscal.
A emissão sempre utiliza um sistema externo, porém esse sistema varia. 
Diferentes sistemas têm diferentes interfaces (recebem diferentes informações
e usando diferentes tecnologias, e retornam a nota em formato diferente). 
O usuário quer sempre a nota em PDF.
